# Harrison Tun — Dinner Table Portfolio

A single-page portfolio built around a top-down restaurant table. Each plate opens a different portfolio section with a seamless zoom transition. The chef opens the contact experience without loading another page.

## Run locally

Open `index.html` directly, or run a local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- `index.html` — the entire portfolio and all section content
- `styles.css` — table scene, food illustrations, modal layouts, and responsive design
- `app.js` — plate zoom transitions, hash navigation, contact interaction, and form submission
- `headshot.jpg` — profile image
- `Harrison_Tun_Resume.pdf` — embedded résumé
